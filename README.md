# Torn Cache Analyzer

A web app for analyzing your [Torn](https://www.torn.com) game cache data.  
**Live site:** https://torn-cache-dashboard.vercel.app/

---

## What it does

Upload your Torn cache JSON file and get a visual breakdown of your in-game data — items, stats, and more — all processed locally in your browser. Your data never leaves your machine.

---

## Running locally

Make sure you have [Node.js](https://nodejs.org) installed, then:

```bash
git clone https://github.com/himlamlolz/torn.git
cd torn
npm install
npm run dev
